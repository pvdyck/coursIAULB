🎯 La Formule Introduction PRS

Structure en 4 paragraphes :

  1. Hook (3-4 phrases) : Problème clinique + Statistiques impactantes
  2. État actuel (4-5 phrases) : Ce qu'on sait + Limites actuelles
  3. Gap (2-3 phrases) : Ce qui manque + Pourquoi c'est important
  4. Objectif (1-2 phrases) : "The purpose of this study was to..."

Longueur idéale : 400-500 mots (1 page manuscrit)

✍️ EXERCICE 4.3 : Introduction Pro

Mission complète (20 min) :

Pipeline:
1. Perplexity : "Stats ASPS 2024 sur [votre sujet]"
2. ChatGPT : "Hook percutant problème clinique"  
3. Claude : "Restructure intro selon formule 4 paragraphes"
4. Gemini : "Vérifie actualité références et ajoute 2025"

Checklist qualité :

Hook accrocheur première phrase
Statistiques récentes (2024-2025)
Gap clairement identifié
Objectif précis et mesurable

🔬 Méthodes : Le Cœur Technique

Standards PRS Methods

Structure obligatoire :

  • Study Design : Type, période, approbation éthique
  • Patient Selection : Critères inclusion/exclusion clairs
  • Surgical Technique : Détails reproductibles + schémas
  • Outcome Measures : PROM validés (FACE-Q, BREAST-Q)
  • Statistical Analysis : Tests utilisés, logiciel, p-value

Erreur fatale : Oublier le numéro d'approbation éthique = Rejet automatique

📈 Résultats : L'Impact Visuel

La règle des 3T

Text

Narratif clair des findings principaux

Tables

Données démographiques et outcomes

Figures

Photos avant/après + graphiques

Règle d'or : Chaque figure/table doit pouvoir être comprise indépendamment du texte

🖼️ EXERCICE 4.5 : Figures Professionnelles

Pipeline création figures PRS

Mission ChatGPT + Code :

"Avec mes données CSV [PROM scores], crée :
1. Figure 1: Box plots pré/post op avec p-values
2. Figure 2: Timeline recovery avec IC 95%
3. Figure 3: Montage photos standardisé 3x3
Style: Arial 12pt, résolution 300 DPI, fond blanc"

Standards photos chirurgie plastique :

  • 📐 Angles standardisés (0°, 45°, 90°, oblique)
  • 🔲 Background uniforme bleu medical
  • 📏 Règle dans le champ
  • 🔒 Yeux masqués barre noire

💭 Discussion : L'Art de l'Interprétation

Structure en 5 parties :

  1. Résumé findings clés (1 paragraphe)
  2. Comparaison littérature (2-3 paragraphes)
  3. Implications cliniques (1-2 paragraphes)
  4. Limites (1 paragraphe honnête)
  5. Futures directions (1 paragraphe visionnaire)

Longueur idéale : 1000-1500 mots (éviter le roman!)

🎯 EXERCICE 4.6 : Discussion Nuancée

Claude Thinking Mode (15 min) :

"Utilise thinking mode pour enrichir ma discussion TFE.
Explore implications non évidentes, biais potentiels,
et applications futures en chirurgie régénérative."

Avant :

"Nos résultats montrent que la technique A est meilleure que B."

Après :

"While our findings suggest superiority of technique A in terms of patient-reported satisfaction (p<0.001), the clinical significance must be interpreted within the context of..."

📚 Références : La Crédibilité

Standards 2025

Règles d'or :

  • 📅 50% références < 5 ans
  • 🌟 Inclure articles fondateurs ET récents
  • 🎯 30-50 références pour article original
  • ✅ Vérifier CHAQUE citation
  • 🔄 Style Vancouver ou journal spécifique

⚠️ Piège mortel :

Les "phantom references" générées par IA = Rejet + Blacklist

📖 EXERCICE 4.7 : Références Parfaites

Workflow sécurisé :

1. Perplexity : "Trouve 10 références clés [technique] 2023-2025"
2. Vérification manuelle PubMed (OBLIGATOIRE)
3. Zotero/Mendeley : Import et formatting auto
4. ChatGPT : "Vérifie cohérence style Vancouver"

Checklist finale :

Chaque référence existe vraiment
Années de publication correctes
Formatting uniforme
Ordre numérique/alphabétique correct

✉️ La Cover Letter Qui Ouvre les Portes

Template gagnant

Structure en 4 paragraphes :

  1. Présentation : Titre + pourquoi ce journal
  2. Innovation : Ce qui est nouveau/unique
  3. Impact : Pourquoi c'est important MAINTENANT
  4. Fit : Alignement avec scope du journal

Longueur : 1 page maximum, personnalisée pour l'éditeur

📮 EXERCICE 4.8 : Cover Letter Persuasive

Prompt Claude optimal :

"Rédige cover letter pour PRS. Mon article porte sur [sujet].
Points uniques : [innovation]. Cible : Editor-in-Chief Rod Rohrich.
Ton : professionnel mais engageant. Maximum 300 mots."

Phrases d'accroche testées :

  • "This manuscript presents the first comparative analysis of..."
  • "We believe this work addresses a critical gap in..."
  • "Our findings challenge the current paradigm of..."

✅ BLOC 3 : Checklist Finale Pré-Soumission

Les 10 Commandements

1. Guidelines journal relues 3 fois
2. Word count respecté (±5%)
3. Figures haute résolution (300 DPI)
4. Supplementary materials prêts
5. Auteurs et affiliations vérifiés
6. Conflits d'intérêts déclarés
7. Funding statement inclus
8. Ethical approval mentionné
9. ICMJE forms signés
10. Suggested reviewers (3-5)

📅 Timeline Post-Soumission

Jour 0 : Soumission
Jour 7-14 : Accusé réception
Semaine 4-8 : Première décision (desk reject ou reviewers)
Semaine 8-12 : Reviews retour
+2 semaines : Révisions soumises
+4 semaines : Décision finale

💬 Répondre aux Reviewers avec IA

Template réponse gagnante :

"Dear Editor and Reviewers,

We thank the reviewers for their constructive comments. 
We have addressed all concerns point-by-point below.

**Reviewer 1:**
Comment: [copier commentaire]
Response: We agree with the reviewer and have [action taken].
This can be found on page X, lines Y-Z.

[Répéter pour chaque commentaire]"

Astuce Claude : "Aide-moi à répondre professionnellement à ce commentaire négatif tout en défendant notre méthodologie"
