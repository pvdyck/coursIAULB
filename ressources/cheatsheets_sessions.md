```

   Ex: Radio → Cancer OUI/NON

   Ex: Groupes de patients similaires

DEEP LEARNING = Analyse complexe multi-couches  
   Ex: IRM cérébrale → Détection tumeur

LLM = Prédiction de texte sur contexte
   Ex: ChatGPT, Claude → Rédaction
```
