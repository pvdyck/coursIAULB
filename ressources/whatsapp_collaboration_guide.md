## 📱 Configuration des Groupes WhatsApp

### 1. Groupe Principal : Annonces

**Nom :** `🎓 ULB IA Med v3 - Annonces`


**Règles :**

- 1 Expert Perplexity (automatisation & rapports)

- `👥 ULB IA Med - Groupe A (Carré 1)`

- `👥 ULB IA Med - Groupe D (Carré 4)`
- `👥 ULB IA Med - Groupe E (Carré 5)`
[Ajuster selon nombre total de participants]

**Description type :**
```
🤝 Groupe de travail collaboratif
💬 Discussion exercices & entraide
🕐 Actif 8h-22h
🏷️ #exercice #outil #question
```

### 3. Groupes Spécialisés par Outil

- `🤖 ULB IA - Experts ChatGPT` (créativité, vision, code)

## 🔄 Workflow Type d'un Exercice

### Phase 1 : Lancement (Lundi 9h)

**Message dans Groupe Principal :**
```
📚 NOUVEL EXERCICE - SEMAINE 3

🎯 Objectif : Comparer diagnostic différentiel sur 3 IA
📝 Instructions : https://drive.google.com/[lien]
⏰ Deadline : Vendredi 17h
🏆 Récompense : Meilleur workflow = 1 mois gratuit

Bonne chance ! 💪
```

### Phase 2 : Travail Collaboratif (Lun-Ven)

**Dans les Groupes de Travail :**

**Étudiant 1 (9h30) :**
```
🔬 Test ChatGPT pour DDx fait !
📊 Score : 8/10 (2 hallucinations)
🖼️ [Screenshot résultats]
💡 Astuce : Prompt en 2 étapes marche mieux
```

**Étudiant 2 (10h15) :**
```
🧪 Comparaison avec Claude :
✅ Plus précis (9/10)
⚡ Plus lent (3min vs 1min)
📎 Résultats complets : [lien Drive]
```

### Phase 3 : Compilation (Jeudi)

**Template Google Sheets à remplir :**
```
| Critère | ChatGPT | Claude | Gemini | Notes |
|---------|---------|--------|--------|--------|
| Rapidité | 5/5 | 3/5 | 4/5 | ChatGPT instantané |
| Précision | 4/5 | 5/5 | 4/5 | Claude sans erreur |
| Sources | 2/5 | 3/5 | 5/5 | Gemini cite tout |
| ... | ... | ... | ... | ... |
```

### Phase 4 : Synthèse (Vendredi 17h)

**Message Groupe Principal :**
```
🏆 RÉSULTATS CHALLENGE SEMAINE 3

🥇 Groupe B - Workflow innovant Claude+Gemini
🥈 Groupe A - Rapidité record avec ChatGPT
🥉 Groupe C - Meilleure détection hallucinations

📊 Tableau complet : [lien]
📝 Workflow gagnant : [lien]

Bravo à tous ! 👏
```

## 🆘 Troubleshooting

### Problèmes Fréquents et Solutions

**"Je ne trouve pas le fichier dans Drive"**
→ Vérifier le dossier de la bonne semaine
→ Utiliser la fonction recherche
→ Demander le lien dans son groupe

**"WhatsApp est saturé de messages"**
→ Activer notifications pour mentions seulement
→ Utiliser la fonction recherche
→ Consulter messages épinglés

**"Je suis perdu dans l'exercice"**
→ Relire instructions dans Drive
→ Demander aide dans son groupe
→ Taguer @formateur si urgent
